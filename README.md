## 🌓 Dual‑World Experience

**One site. Two dimensions.**

- **⌨️ Logic & Code** — Android (Kotlin/Java), Python, clean architecture.  
  Powered by neon rain, a glowing tech grid, and live GitHub repositories.

- **✦ Visual & Art** — Branding, UI/UX, packaging.  
  Floating in a dreamy particle field with a draggable portfolio carousel.

Toggle between worlds with a smooth pill navigation – the entire UI changes its accent and atmosphere instantly.

---

## 🧬 Core Features

- 🌗 **Dark & Light themes** — system‑aware, hand‑crafted colour palettes
- 🖱️ **Custom cursor** with lag‑ring physics (desktop)
- 🌧️ **Dynamic backgrounds** — rain engine, floating particles, drifting orbs
- 📜 **Certificate lightbox** — click any credential to zoom in
- 🎠 **3D carousel** — responsive, draggable, keyboard‑accessible
- 📦 **Live GitHub repos** — fetched via API with shimmer skeletons
- ♿ **Accessible** — ARIA labels, keyboard navigation, prefers‑reduced‑motion support
- 📱 **Mobile‑first touches** — glass‑effect blur, swipe‑lock between worlds, floating theme toggle

## 🛠️ Tech Stack

| Layer        | Technology                                   |
| ------------ | -------------------------------------------- |
| Structure    | Semantic HTML5, JSON‑LD structured data      |
| Styling      | CSS Custom Properties, `backdrop-filter`, Grid, Flexbox, `clamp()` |
| Animation    | CSS `@keyframes`, `requestAnimationFrame`, Intersection Observer |
| Data         | GitHub REST API (public repos)               |
| Fonts        | Syne (display) + DM Sans (body) via Google Fonts |
| Hosting      | GitHub Pages                                 |
